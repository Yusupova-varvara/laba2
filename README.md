Задание№1
2. (Сжатие строки): Дана строка S, необходимо преобразовать
строку в её сжатую форму, заменив последовательности повторяющихся
символов на сам символ и количество его повторений; если сжатая версия
оказывается не короче исходной строки, следует вернуть исходную строку.
Пример 1: S = “aabcccccaaa”, Результат: “a2b1c5a3”
Пример 2: S = “aabbcc”, Результат: “aabbcc”
Пример 3: S = “aaaaaa”, Результат: “a6”

Задание №2
2. (Панграмма): Панграмма, или разнобуквица – короткий текст,
использующий все или почти все буквы алфавита, по возможности не повторяя
их. Учитывая строку S, содержащую только строчные буквы латинского
алфавита, вернуть Yes, если S – панграмма, и No – если нет.
Пример 1: S = “the five boxing wizards jump quickly”, Результат:
Yes
Пример 2: S = “cozy sphinx waves quart jug of bad milk”, Результат:
Yes
Пример 3: S = “the quick brown fox jumps over the lazy dog”,
Результат:
Yes

2. На вход подаётся N натуральных чисел, необходимо вывести все
числа, которые составлены из цифры только одного вида (например, 111, 44
и т. д.).
Пример 1: N = 4, 33 12 11 245, Результат: 33 11
Пример 2: N = 3, 1 222 4, Результат: 222
